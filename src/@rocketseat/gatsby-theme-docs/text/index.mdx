# Lierno bot
Lierno bot es un bot de Discord para TTRPG naturalmente integrado con [la aplicación de Lierno](https://lierno.netlify.app/) para su uso en rol a través de chat.

## Tiradas de dados avanzadas
Con una librería de dados integrada, Lierno bot soporta casi todos los tipos de tiradas estándar de TTRPG. Ventaja, desventaja, críticos, etc.

## Integración con fichas de personaje
Lierno bot puede leer fichas de personaje creadas a través de [la aplicación de Lierno](https://lierno.netlify.app/), generando automáticamente muchos tipos de ataques, tiradas de habilidad, y tiradas de salvación.
